# Image Assets

Place the following images in this folder.
They are referenced throughout the app via `/images/<filename>`.

| File | Description | Usage |
|---|---|---|
| `robot-idle.png` | Friendly blue robot (Bleep), neutral pose | Home page & lesson header |
| `robot-success.png` | Bleep happily eating toast | Shown on correct answer |
| `sticker-algorithm.png` | Gold star medal with "1" on it | Success sticker overlay |

## Generating with DALL-E (ChatGPT)

Use these prompts:

**robot-idle.png**
> "Cute friendly blue cartoon robot with big eyes and round body, waving, flat vector style, white background, suitable for children aged 6–12"

**robot-success.png**
> "Cute friendly blue cartoon robot happily eating a slice of toast, celebrating, flat vector style, white background, children's illustration"

**sticker-algorithm.png**
> "Gold star medal sticker with the number 1 and the word Algorithm, shiny, celebration style, flat vector, transparent background"
